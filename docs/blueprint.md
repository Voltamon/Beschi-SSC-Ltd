# **App Name**: Ethereal Visions

## Core Features:

- Image Trail: Display a captivating trail of images starting with the provided URL and Gemini-generated suggestions.
- Liquid Effect Transition: Integrate a React component for a mesmerizing liquid effect transition between pages.
- Dynamic Image Loading: Load and display images using the LiquidBackground component with dynamic loading from provided URLs and assets.
- Interactive Liquid Simulation: Implement interactive liquid simulation effects via manipulation of metalness, roughness, and displacement scale. Adjust rain.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to evoke mystery and sophistication, in line with the ethereal theme.
- Background color: Very light Lavender (#F0F8FF) for a soft, dreamlike quality.
- Accent color: Electric Violet (#8F00FF) for highlighting interactive elements and creating visual contrast.
- Body and headline font: 'Belleza', a humanist sans-serif suitable for fashion and design.
- Use lucide-react icons to represent interactive elements. Icons should have a line art style and a thin stroke width for elegance.
- Ensure the image trail and liquid effect animations cover the full viewport for an immersive experience.
- Implement smooth transitions and animations when switching between images and applying the liquid effect. Optimize liquid-effect-animation.tsx to run without performance issues.
